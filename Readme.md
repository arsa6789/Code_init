## Getting Started

Each project folder contains its own set of files. To view a project:
1. Navigate to the project folder
2. Open the `index.html` file in a web browser
3. Interact with the project elements

## Learning Focus

These projects cover a wide range of front-end development concepts including:
- HTML form handling and semantic markup
- CSS animations, layouts, and effects
- JavaScript DOM manipulation and interactivity
- Bootstrap framework usage
- TypeScript integration
- Firebase backend integration
